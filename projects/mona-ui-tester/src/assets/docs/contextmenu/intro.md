# Context Menu

**Selectors:** `mona-contextmenu` · `mona-contextmenu-item` · `mona-contextmenu-checkbox-item` · `mona-contextmenu-radio-group` · `mona-contextmenu-radio-item` · `mona-contextmenu-group` · `mona-contextmenu-separator`

`ContextMenuComponent` attaches a popup menu, built from content-projected menu item components, to a target element and opens it on right-click or a keyboard shortcut.
